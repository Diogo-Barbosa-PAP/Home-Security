Linguagens

Para a criação deste website, foi utilizado html, css e javascript. Para o html e css foi consultado o website W3Schools e para o javascript foi consultado predominantemente o website StackOverflow.



O software utilizado para o desenvolvimento do website foi o Visual Studio Code, que possui várias vantagens relativamente a outros programas de edição de texto, sendo a maior delas a possibilidade de editar o texto num live server (ou seja ao salvar as alterações no Visual Studio Code automaticamente salva na web e é possivel ver os resultados em tempo real sem necessidade de dar refresh ao browser).



Obviamente o website possui várias falhas, a não utilização de PHP é um fator crucial para a desvalorização do website, porém penso que dentro das minha capacidades consegui criar um website que apresenta a PAP de uma forma objetiva.



O código utilizado será apresentado em cada dropdown-menu existente na barra de navegação e está neste link GitHub disponível para download.
