Javascript (Js)

O JavaScript é uma linguagem de programação do lado cliente, ou seja, é processada pelo próprio navegador. Com o JavaScript podemos criar efeitos para páginas na Web, além de podermos proporcionar uma maior interatividade com os usuários.

O JavaScript é uma linguagem orientada a objetos, ou seja, ela trata todos os elementos da página como objetos distintos, facilitando a tarefa da programação.

O Javascript utilizado neste website é extremamente curto, pois foi apenas utilizado para a criação da animação da barra de navegação responsiva (a transição das 3 barras transversais para um género de "X").
