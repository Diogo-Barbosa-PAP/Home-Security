HyperText Markup Language (HTML)

HTML significa Hypertext Markup Language. Esta linguagem permite que os usuários criem e estruturem seções, parágrafos, cabeçalhos e links para páginas da internet ou aplicações.

O HTML não é uma linguagem de programação, isso significa que não pode ser usado para criar funcionalidades dinâmicas. Entretanto, possibilita a organização e formatação de documentos, similar ao Microsoft Word.
